# Jovial for Sublime Text 3
Syntax highlighting for the Jovial/J73 standard, based on MIL-STD-1589C. 

## Installation
Download and unzip the JovialJ73 folder, then place the folder in your operating system's Sublime Text 3 configuration folder. 

**Linux directory:**
```
~/.config/sublime-text-3/Packages
```

I'll add in the Windows directory location in a future update.
